# Kroyii

An AI-powered onchain agent built on Ethereum.

## Live App

[https://kroyii.vercel.app](https://kroyii.vercel.app)

## Built With

- [Scaffold-ETH 2](https://scaffoldeth.io)
- NextJS
- Venice AI
- RainbowKit
- Wagmi / Viem
- Pimlico (ERC-7715 permissions)
- Hardhat

## Getting Started

Install dependencies:

```bash
yarn install
```

Start the app:

```bash
yarn start
```

Visit `http://localhost:3000`
